Optional Brain Teaser

If you feel like giving your brain a little workout, try solving this simple problem. It’s not mandatory, but it’s a fun way to show off your skills!

Task: Write a Python function that takes a list of numbers and returns the list sorted in ascending order, but with all even numbers appearing before any odd numbers.


def sort_even_odd(numbers):
    # Separate even and odd numbers
    evens = [n for n in numbers if n % 2 == 0]
    odds = [n for n in numbers if n % 2 != 0]
    
    # Sort the two lists
    evens.sort()
    odds.sort()
    
    # Combine the sorted lists, evens first
    return evens + odds

# Example usage:
numbers = [5, 3, 2, 8, 1, 4]
sorted_numbers = sort_even_odd(numbers)
print(sorted_numbers)  # Output should be [2, 4, 8, 1, 3, 5]
